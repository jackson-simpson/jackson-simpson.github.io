Differences between Relative, Absolute and Fixed Positioning

Static Positioning

Static positioning is the basic, defult property that all elements have. Without any additional position properties, the elemental will act statically. This means it will generally be moving down the working axis of the page, typically vertically. This doesn't give us a lot to work with, so we add other positioning properties to class and ID elements to move them around the page. Relative Positioning

Relative positioning is moving an element relative to its position in the documents flow by the use of coordinates.

Thats a lot of jargon, what does that mean? Essentially:

  1. An Element is the thing you are moving, usually something like a division.
  2. Document Flow is how elements arrange themselves down a page, this is by defult done in blocks down a page, and done by inline elements across a page.
  3. Coordinates are the instructions for the positioning; i.e. 'Top: 30px;'
  4. and'Relative to its Position' means where it already is on your page

This means the 'relative' property does not remove the element from document flow, and that it is displayed position is the only thing that changes. Other elements around will not move up to replace it if say the element moved down the page. It can also move to cover other elements based on the coordinates you use. Note also that coordinates are based on the sides of the element, so if you are moving your element with the coordinates 'Top: 30px;' the element will move down 30px.

Fixed Positioning

The last type of positioning we will be discussing in this blog-post is Fixed positioning. This is when the position of the element is not fixed to any other element, rather it is fixed to its position on the display. Much like absolute positioning, it will ignore the normal document flow, as well as any parent elements.

Note the position of the scroll-bar. Despite going down the page, the element has not moved. This can be useful for anchoring messages or images to certain parts of your website.

And those are basically the differences between these three positioning properities. There are others, such as 'sticky' positioning and they can also be very helpful tools.
